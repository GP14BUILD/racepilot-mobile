## üí° Tips

### Finding Information Fast

**Use Ctrl+F** (or Cmd+F on Mac) to search within documents for:
- Specific error messages
- Feature names
- Commands
- Settings

**Check the right doc**:
- **User problems** ‚Üí USER_GUIDE.md
- **Build problems** ‚Üí GETTING-STARTED.md or ANDROID_STUDIO_GUIDE.md
- **Code questions** ‚Üí README.md

### Getting Unstuck

1. **Check troubleshooting sections** in relevant docs
2. **Search for your error message** (Ctrl+F)
3. **Read from the beginning** of the relevant guide
4. **Check you followed all steps** exactly
5. **Email support** if still stuck
